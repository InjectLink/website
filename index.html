IDC-NG

Contrast Injector Communication Module

Request information Documentation

IDC-NG connects injection systems to hospital DICOM infrastructure. It captures injection data over CANopen and securely publishes structured reports to PACS, while enabling patient association via DICOM Worklist through an intuitive web interface.

Designed for Modern Radiology

CANopen CiA-425 Native

Real-time acquisition of injection parameters using CiA-425 compliant CANopen communication.

DICOM Structured Reports

Automatic generation and transmission of injection reports via C-STORE.

Worklist Integration

Retrieve Modality Worklist and associate patients directly from the embedded web interface.

Simple. Secure. Interoperable.

IDC-NG acts as a seamless bridge between injector devices and hospital IT systems.
